# Naga Mas ERP

Internal ERP dashboard untuk dealer Honda PT Nagamas Mitra Sejati — wilayah Tegal & Brebes.

## Fitur
- Dashboard overview penjualan & KPI
- Inventori & stok motor
- Sales & CRM
- Workshop AHASS
- Keuangan & laporan
- Manajemen SDM & HR

## Tech Stack
- React 18 + Vite
- Recharts (grafik)
- Lucide React (ikon)

## Cara Jalankan Lokal

```bash
npm install
npm run dev
```

## Deploy ke Vercel

1. Push repo ke GitHub
2. Import repo di [vercel.com](https://vercel.com)
3. Vercel otomatis deteksi Vite — klik **Deploy**
